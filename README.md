# groceroo

## How to run
#### 1) Generate bundle for backend to serve:
`cd frontend`

`npm i`

`npm run dev`

#### 2) Start backend server:
Go to root directory `cd ..` and run 
`pipenv shell`

`pipenv install -r requirements.txt`

`python manage.py runserver`
